Introduction

Tailwind CSS is a utility-first CSS framework that allows you to style elements directly within your HTML using utility classes. This approach helps you create custom designs faster without writing custom CSS. It offers a highly customizable configuration and integrates with modern build tools like Webpack, PostCSS, and others.

Utility Classes

Tailwind’s core feature is its utility classes, which are single-purpose classes that do one thing, like text-center, p-4, or bg-blue-500. These can be combined to create complex designs without writing additional CSS.

Example

<div class="bg-blue-500 text-white p-4 rounded-lg" >Button</div>

Responsive Design

Tailwind includes responsive utilities by prefixing utility classes with breakpoints like

  • sm
  • md
  • lg and
  • xl

This enables you to design mobile-first and adjust styles for larger screens.

Example

<div class="p-4 sm:p-6 md:p-8">Responsive Padding</div>

Components

Unlike frameworks like Bootstrap, Tailwind doesn’t come with pre-built components. Instead, you build components using utility classes, giving you complete control over the design.

Example

<div class="bg-gray-200 p-6 rounded-md shadow-md"> <h2 class="text-xl font-semibold">Card Title</h2> <p class="text-gray-700">Card content goes here. </p></div>

Plugins

Tailwind has a rich ecosystem of plugins that extend its functionality, including forms, typography, aspect ratio, and more. You can install plugins to add additional utilities or components.

Example

npm install @tailwindcss/forms

Browser Compatibility

Tailwind CSS is compatible with all modern browsers (Chrome, Firefox, Safari, Edge, etc.) and supports older versions of Internet Explorer (with some limitations). Tailwind also uses modern CSS features like Flexbox and Grid, which should be considered when working with older browsers.
